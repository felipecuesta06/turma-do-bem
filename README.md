# turma-do-bem
Sistema CLI em Python para a ONG Turma do Bem, focado na gestão de pacientes e dentistas voluntários. O software realiza o cadastro completo com validação de CPF, registro de tratamentos e geração de relatórios de atendimento. Utiliza lógica modular, listas de dicionários e filtros de busca, garantindo integridade e organização dos dados.
